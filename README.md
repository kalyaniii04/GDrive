Description

GDrive is a full-stack file storage web application inspired by cloud drive platforms. It allows users to upload, store, and manage files through a clean server-side rendered interface, focusing on backend integration and file handling.

This project demonstrates practical use of authentication-ready architecture, cloud storage integration, and secure file management.

Features

File upload and storage functionality

Server-side rendering using EJS

Express.js backend with structured routing

MongoDB integration for data management

Cloud-based deployment on Render

Tech Stack

Frontend: EJS, HTML5, CSS3, JavaScript

Backend: Node.js, Express.js

Database: MongoDB, Mongoose

Storage: Cloudinary

Deployment: Render
